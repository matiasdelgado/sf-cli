# Usage
<!-- usage -->
```sh-session
$ npm install -g sf
$ sf COMMAND
running command...
$ sf (-v|--version|version)
sf/0.0.0 darwin-x64 node-v10.13.0
$ sf --help [COMMAND]
USAGE
  $ sf COMMAND
...
```
